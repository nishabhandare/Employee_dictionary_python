# Employee Dictionary Management

This is a Python dictionary project using user input.

## Features
- Create employee dictionary
- Add key and value
- Update dictionary values
- Check key existence
- Display dictionary keys and values
- Find dictionary length
- Remove dictionary key using pop()

## Concepts Used
- Dictionary
- Loop
- Conditional Statements
- User Input
- if-else
- Dictionary methods

## Sample Operations
- Add employee details
- Update employee name
- Add age key
- Remove age key

## Language Used
- Python
